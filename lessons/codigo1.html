Un ejemplo básico para comenzar con JavaScript.

Ejemplo 3: Funciones y Bucles

Estructuras de control y funciones básicas.

// Bucle for
for (let i = 0; i < 5; i++) {
    console.log(`Iteración ${i}`);
}

// Función con parámetros
function calcularArea(base, altura) {
    return base * altura;
}

let area = calcularArea(10, 5);
console.log(`El área es: ${area}`);

Ejemplo 4: Manipulación del DOM

Interactuando con elementos HTML desde JavaScript.

// Seleccionar elementos
const boton = document.querySelector('#miBoton');
const texto = document.querySelector('#miTexto');

// Agregar evento
boton.addEventListener('click', () => {
    texto.textContent = '¡Texto cambiado!';
    texto.style.color = '#3B82F6';
});

// Crear nuevo elemento
const nuevoDiv = document.createElement('div');
nuevoDiv.className = 'card';
nuevoDiv.textContent = 'Nuevo contenido';
document.body.appendChild(nuevoDiv);
