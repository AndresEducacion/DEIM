Básicos de la programación en p5.js

Conceptos fundamentales para aprender a programar de forma visual y creativa.

1. ¿Qué es la programación?

Programar es dar instrucciones a un ordenador para que realice tareas. Estas instrucciones se escriben en un lenguaje que la computadora puede entender.

El código se ejecuta de arriba hacia abajo. El orden sí importa.

  • Automatizar procesos y crear aplicaciones, juegos y arte.
  • Un lenguaje de programación tiene reglas propias.
  • Usaremos JavaScript con p5.js (visual y creativo).

2. Variables

Una variable es un contenedor donde guardamos información que puede cambiar mientras el programa está funcionando.

Para crear una variable usamos let seguido del nombre. Su valor puede cambiar en cualquier momento. 

let x = 50;  // Guardamos un número en la variable x

function draw() {
  x = x + 1;  // Actualizamos el valor para mover un objeto
}

Gracias a las variables, nuestros programas pueden ser dinámicos y cambiar con el tiempo.

3. Estructuras de control (Condicionales)

Las estructuras if y else permiten que el programa tome decisiones basadas en condiciones.

Ejemplo if / else

let x = 0;

function draw() {
  x = x + 1;

  if (x > 200) {
    fill(255, 0, 0);
  } else {
    fill(0, 0, 255);
  }

  circle(x, 200, 50);
}

if…else if

if (x < 100) {
  // primera condición
} else if (x < 200) {
  // segunda condición
} else {
  // si ninguna se cumple
}

5. Bucles

Un bucle permite repetir acciones múltiples veces sin escribirlas una por una.

Ejemplo con for

function draw() {
  background(220);

  // Bucle que dibuja líneas verticales
  for (let x = 0; x < width; x = x + 20) {
    line(x, 0, x, height);
  }
}
